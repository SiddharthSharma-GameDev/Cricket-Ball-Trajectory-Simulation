Cricket Ball Trajectory Simulation

A Unity-based cricket ball trajectory simulation created as a technical assignment.

Features

- Swing delivery mode
- Spin delivery mode
- Movable bounce marker using WASD controls
- Left/right delivery variation
- Strength meter for delivery intensity
- Trajectory preview line before bowling
- Smooth ball bounce and post-bounce movement
- Clean UI controls for bowling interaction

Controls

- WASD: Move bounce marker
- Swing: Select swing delivery
- Spin: Select spin delivery
- Change Side: Switch left/right variation
- Bowl: Deliver the ball

Built With

- Unity 6
- C#
- WebGL and Windows Build support

Project Structure

Assets:
Contains game assets, scenes, scripts, materials, and UI.

Packages:
Contains Unity package dependencies.

ProjectSettings:
Contains Unity project configuration.
